# Play Store Assets Checklist

## Required Assets for Play Store Submission

### App Icons (Already have ✅)
- [x] App icon (512x512 PNG) - Use your existing icon.png
- [x] Adaptive icon - Use your existing adaptive-icon.png

### Screenshots (Need to create)
- [ ] Phone screenshots (2-8 images, 16:9 or 9:16 aspect ratio)
  - Minimum resolution: 320px
  - Maximum resolution: 3840px
  - At least 2 screenshots required

### Feature Graphic
- [ ] Feature graphic (1024x500 PNG)
  - Showcases your app prominently
  - Used in Play Store promotions

### Store Listing Content
- [ ] App title: "Bhav Bhakti" (max 50 characters)
- [ ] Short description (max 80 characters)
- [ ] Full description (max 4000 characters)
- [ ] App category: Lifestyle or Entertainment
- [ ] Content rating questionnaire
- [ ] Privacy Policy URL (required for apps that handle user data)
- [ ] Contact information

### Additional Requirements
- [ ] Target API level 34 (Android 14) or higher
- [ ] App bundle signed and uploaded
- [ ] Release notes
- [ ] Pricing (Free/Paid)
- [ ] Countries/regions for distribution
